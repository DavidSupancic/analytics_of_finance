- Prvo kreirajte i aktivirajte Python virtualnu okolinu: 

python3-m venv .venv source .venv/bin/activate 

- Nakon toga, dohvatite i instalirajte potrebne biblioteke: 

pip install-r requirements.txt

- Ako vam nedostaje neka od biblioteka, možete je instalirati pomoću pip-a:

pip install naziv_biblioteke    

- Popunite prazna mjesta u bilježnici.
